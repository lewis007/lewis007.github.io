## Jekyll Quick Start

```bash
~ $ gem install bundler jekyll
~ $ jekyll new my-awesome-site
~ $ cd my-awesome-site
~/my-awesome-site $ bundle exec jekyll serve
# => Now browse to http://localhost:4000
```

## 本地运行

```bash
~ $ git clone https://github.com/lewis007/lewis007.github.io.git
~ $ cd lewis007.github.io
~/lewis007.github.io $ bundle exec jekyll serve
# => Now browse to http://localhost:4000
```
